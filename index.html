Utility Pages

A small collection of free, browser-based tools. No sign-up, nothing leaves your browser.

  • ☷ Text Comparison Compare two texts side by side and highlight every difference, with options to ignore case, spaces, line feeds and escape sequences.
  • ¶ Line Break Converter Turn line breaks into commas (or any separator), convert in either direction, and optionally wrap each item with a template like '{text}'.
  • M↓ Markdown Editor Write Markdown with a live, styled preview beside it — headings, tables, code blocks and more, with synchronized scrolling. Paste rich text into the preview to turn it into Markdown.
